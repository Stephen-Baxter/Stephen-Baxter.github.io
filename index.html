Projects List

     The tabs below can be used to limit the search results, row one are the classes, row two are the languages, and row three are the interfaces types.

Class:
     Unique are projects that are created to perform a specified action or purpose and have their own names.
     Generic are projects that are created to provide proof of coding abilities and may have names used my other projects.

Language:
     HTML5 are projects created using the HTML5, JavaScript, and CSS languages.
     C++ are projects created using the C pluses pluses language.
     C# are projects created using the C sharp language.
     VB are projects created using the visual basic.net language.

Interface:
     GUI are projects that use a graphic user interface for the user to interact with the program.
     CLI are projects that use a command line interface for the user to interact with the program.
     NVI are projects that have no visual interface for the user to interact with the program.
